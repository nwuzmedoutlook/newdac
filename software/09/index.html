CiteSpace文献计量软件的安装和启动（Win32位、64位）

第一步：配置Java环境


1、根据自己电脑情况，下载对应版本Java，32位电脑下载x86版，64位电脑下载x64版，链接如下：

Java SE Runtime Environment 8​www.oracle.com/java/technologies/javase-jre8-downloads.html


2、安装Java之前，在除C盘以外的英文目录下新建一个Java文件夹
安装时，勾选“更改目标文件夹”，点击“安装”后，选择之前新建的Java文件夹；之后自动安装就好。




第二步：安装CiteSpace软件


1、下载CiteSpace安装启动器


64位电脑请点击下部链接，在官网下载，一般建议下载后缀为.7z的安装文件，如下图所示：

https://sourceforge.net/projects/citespace/files/5.7.R5%20%28expires%20December%2031%2C%202021%29/​sourceforge.net/projects/citespace/files/5.7.R5%20%28expires%20December%2031%2C%202021%29/

2015年之后，32位的CiteSpace不再更新，软件作者陈朝美博士的博客上的下载链接已经失效，但目前还有部分同学仍在使用32位电脑，所以在此我提供3.8.R1.32-bit.1.29.2014版的安装包，链接如下，自行下载

3.8.R1.32-bit.1.29.2014.zip - 蓝奏云​nwuzmed.lanzouw.com/ixSCCn2i3kb

2、安装CiteSpace
64位安装包解压之后，直接双击StartCiteSpace_Windows.bat文件，随后出现命令窗口。输入数字2，代表使用中文安装。窗口先显示Initializing CiteSpace，进度条达到100%后点击“确定”；

随后会显示Downloading WOS Demo Files，这都是在联网下载必要文件，最终会在C盘多出一个.citespace文件夹，位置一般是C:\Users\电脑品牌英文或Administrator，例如我的是C:\Users\asus\.citespace。

建议选择夜间0点至早上8点前安装，避免网络拥堵，否则可能下载卡顿一两个小时甚至更长时间。必要文件下载完之后会自动启动软件，初始界面如下：


点击底部Agree按钮（此处截图失误，没截到），进入下图界面，即可操作啦！

32位安装时，同样是双击StartCiteSpace_Windows.bat文件，随后出现命令窗口，输入数字2，会自动安装完毕并启动。点击底部Agree后可进入操作界面。

需要注意的是32位软件早已经停止更新，目前的安装包虽然可以装上，但某些功能欠缺或者不能分析较新的文献，请谨慎使用。

第三步：启动Cite Space软件

安装完毕后，关闭软件。再次启动时，依旧是找到原来CiteSpace安装包路径中的StartCiteSpace_Windows.bat，双击，输入2即可启动啦！


最后


网传无法顺利安装的同学，可以将别人已经下载好的.citespace文件夹直接放在C:\Users\电脑品牌英文或Administrator 的位置，但前提是已经安装好适合自己电脑的Java，以及CiteSpace安装启动器（需要里面的StartCiteSpace_Windows.bat文件）。具体是否可行，诸君自己尝试。我这里附上已经成功安装过的.citespace文件，点击链接下载。


.citespace.zip - 蓝奏云​nwuzmed.lanzouw.com/ikmtfn2pp0f
