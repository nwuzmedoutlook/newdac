# 01. JSX简介

## 什么是JSX？

```js
const element = <h1>Hello, world!</h1>;
```

以类似模板语法的方式描述UI的一种语法。它是JS的**语法糖**，**本质上也是JS**。可以在其中使用**变量或表达式、函数等**。
